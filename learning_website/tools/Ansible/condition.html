Condition 

=> Whenever we have different different scenarios, we put conditions accoring to scenarios. 

When statement
Some times you want to skip a particular command on particular node.

---
- name: conditional playbook
  hosts: server
  user: ansible
  become: yes
  connection: ssh
  tasks:
          - name: Install apache pkg for debian family
            command: apt-get -y install apache2
            when: ansible_os_family == "Debian"

          - name: Install apache pkg for redhat
            yum:
                    name: httpd
                    state: present
            when: ansible_os_family == "RedHat"
