Loops

Sometimes you want to repeat a task multiple times. In computer programming this is called loops. Common Ansible loops include changing ownership on
several files and/or directories with the file module, creating multiple users with user module, and repeating a polling step untill certain result
is reached.

---
- hosts: server
  user: ansible
  become: yes
  connection: ssh
  tasks:
          - name: add list of users in my nodes
            user: name='{{item}}' state=present
            with_items:
                    - dadu
                    - papa
                    - mummy
                    - nidhi
